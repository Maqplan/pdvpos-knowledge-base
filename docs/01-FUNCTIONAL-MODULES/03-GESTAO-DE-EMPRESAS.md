# PDVPOS ERP

> Official Product Documentation

# 03 - Gestão de Empresas

---

# Objetivo

O módulo **Gestão de Empresas** concentra as configurações estruturais da plataforma, permitindo administrar empresas, filiais e parâmetros gerais que influenciam o comportamento de todo o ERP.

Além das informações cadastrais, este módulo adapta o sistema ao segmento de atuação da empresa, habilitando funcionalidades específicas conforme o ramo de atividade.

---

# O que este módulo resolve

O módulo foi desenvolvido para permitir que empresas de diferentes segmentos utilizem a mesma plataforma sem complexidade desnecessária.

Entre os principais objetivos estão:

- Centralizar os dados cadastrais da empresa.
- Administrar grupos empresariais.
- Configurar comportamentos específicos por segmento.
- Definir parâmetros fiscais e operacionais.
- Controlar empresas e filiais em uma única plataforma.

---

# Recursos Disponíveis

| Recurso | Descrição |
|----------|-----------|
| Cadastro de Empresas | Gerenciamento das informações cadastrais da empresa. |
| Multiempresa | Administração de múltiplas empresas em uma única plataforma. |
| Controle por Filial | Configuração independente por unidade operacional. |
| Configuração por Segmento | Habilitação automática de funcionalidades conforme o ramo de atividade. |
| Parâmetros Gerais | Configurações que determinam o comportamento global do ERP. |

---

# Gestão Multiempresa

O PDVPOS ERP permite controlar diversas empresas utilizando uma única plataforma.

Cada empresa mantém sua independência operacional, podendo possuir:

- Produtos habilitados.
- Preços próprios.
- Estoques próprios.
- Clientes específicos.
- Usuários específicos.
- Vendedores específicos.
- Depósitos próprios.
- PDVs próprios.

Essa arquitetura permite administrar grupos empresariais preservando a autonomia de cada operação.

---

# Controle por Filial

As filiais funcionam como unidades operacionais independentes.

O sistema permite controlar diversas informações por filial, incluindo:

- Produtos habilitados para venda.
- Estoque individual.
- Preços diferenciados.
- Clientes específicos.
- Vendedores específicos.
- Usuários autorizados.
- Operadores de caixa.
- Depósitos.

Todas essas informações permanecem sincronizadas com os PDVs vinculados à respectiva filial.

---

# Configuração por Segmento

Um dos diferenciais do PDVPOS ERP é adaptar a plataforma ao segmento de atuação da empresa.

Ao definir o ramo de atividade, funcionalidades específicas podem ser habilitadas automaticamente.

## Segmentos suportados

- Comércio
- Supermercado
- Padaria
- Alimentação
- Vestuário
- Calçados
- Prestação de Serviços
- Clínica Médica
- Revenda de Combustíveis

---

# Recursos Específicos por Segmento

## Vestuário e Calçados

Habilita o gerenciamento de produtos por grade.

Permite:

- Grade de produtos.
- Variações.
- Estoque individual.
- Código de barras individual.
- Preço individual por variação.

---

## Prestação de Serviços

Habilita o módulo de **Ordens de Serviço**.

Permite controlar:

- Instalações.
- Montagens.
- Serviços técnicos.
- Agenda de atendimento.
- Técnicos responsáveis.

---

## Clínica Médica

Disponibiliza recursos específicos para clínicas de oftalmologia.

Entre eles:

- Cadastro de receituários.
- Grau.
- Curvatura.
- DNP.
- DP.
- Esférico.
- Cilíndrico.
- Eixo.
- Adição.
- Observações.

---

## Revenda de Combustíveis

Disponibiliza informações fiscais específicas para combustíveis.

Inclui:

- Código ANP.
- Percentual de GLP.
- GNN.
- GNI.
- Dados exigidos para emissão fiscal.

---

# Benefícios

A configuração por segmento oferece diversas vantagens:

- Interface mais simples.
- Menor quantidade de configurações desnecessárias.
- Implantação mais rápida.
- Melhor experiência para o usuário.
- Recursos especializados quando necessários.

---

# Integração com Outros Módulos

As configurações realizadas neste módulo influenciam diretamente diversos recursos da plataforma.

Entre eles:

- Produtos.
- Serviços.
- Documentos Fiscais.
- Ordem de Serviço.
- Estoque.
- Financeiro.
- Usuários.
- Relatórios.

---

# Diferenciais

O módulo Gestão de Empresas permite que o mesmo ERP atenda empresas de diferentes segmentos sem necessidade de versões distintas do sistema.

Ao habilitar funcionalidades específicas conforme o ramo de atividade, o PDVPOS ERP reduz a complexidade operacional e mantém uma experiência de uso mais adequada à realidade de cada negócio.

---

# Benefícios para o Cliente

- Plataforma preparada para diferentes segmentos.
- Gestão centralizada de grupos empresariais.
- Controle independente por filial.
- Configuração simplificada.
- Crescimento sem necessidade de troca de ERP.
